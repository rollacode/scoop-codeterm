# Scoop Bucket for CodeTerm

AI-first cross-platform terminal multiplexer.

## Install

```powershell
scoop bucket add codeterm https://github.com/rollacode/scoop-codeterm
scoop install codeterm
```

## Update

```powershell
scoop update codeterm
```

## Uninstall

```powershell
scoop uninstall codeterm
```
